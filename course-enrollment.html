학생 수강 등록

수업 관리의 `학생 수강 등록` 버튼이나 학생 관리의 `수업 수강 등록` 버튼을 통해 진입하는 공용 등록 화면입니다. 수업 기준 정보는 자동으로 불러오되, 학생이 실제로 몇 회차부터 들어가며 얼마를 청구할지 개별 수정할 수 있습니다.

학생 관리로 수업 관리로

진입 방식 선택

같은 등록 화면이지만, 어디서 들어왔는지에 따라 선선택된 정보가 달라집니다. 학생 기준이면 학생이 먼저 고정되고, 수업 기준이면 수업 정보가 먼저 고정됩니다.

학생 기준 진입 수업 기준 진입 회차별 청구 수정 가능
경로 A

학생 관리에서 수업 수강 등록

학생 테이블의 `수업 수강 등록` 버튼을 누르면, 해당 학생 정보가 먼저 고정된 상태로 열립니다. 이후 수업을 선택하고 회차/청구 조건을 조정합니다.

경로 B

수업 관리에서 학생 수강 등록

수업 테이블의 `학생 수강 등록` 버튼을 누르면, 해당 수업 정보와 기준 회차/금액이 먼저 올라옵니다. 여기서 등록할 학생을 추가 선택해 마무리합니다.

등록 단계

1. 진입 정보 2. 학생 / 수업 선택 3. 수강 회차 설정 4. 청구 / 할인 설정 5. 등록 확인

Step 1. 진입 정보

현재 화면이 어떤 경로로 열렸는지와, 자동으로 채워진 기준 정보를 먼저 확인합니다.

현재 예시는 `수업 관리 > 수업 테이블 > 학생 수강 등록` 흐름으로 진입한 상태입니다. 선택 수업의 총 회차, 기준 수강료, 기타 청구 항목은 수업 등록 시 입력한 값으로 자동 반영됩니다.
선택 수업 2025 SAT class / 2025-26 Winter
T.김성명 · 토/일 16:00 - 18:00 · 오프라인
기준 수업 정보 총 12회 · 회당 100,000원 기준
수업 등록 시 시험 SAT / AP 연동

Step 2. 학생 / 수업 선택

학생 기준 진입이든 수업 기준 진입이든, 최종적으로는 `어느 학생이 어떤 수업에 등록되는지`를 명확히 고정해야 합니다.

Step 3. 수강 회차 설정

수업 등록 시 만들어진 기준 회차를 자동으로 가져오되, 학생이 실제로 몇 회차부터 들어가고 총 몇 회를 수강하는지에 따라 수강 회차와 청구 금액을 개별 계산합니다.

회차별 수강 반영

Step 4. 청구 / 할인 설정

수업 등록 시 기준된 금액을 자동으로 가져오되, 실제 수강 회수와 개별 할인에 따라 최종 청구 금액을 수정합니다.

기준 청구 항목

개별 할인 / 조정

금액 할인과 비율 할인은 함께 사용할 수 있으며, 최종 금액에서 순차 반영됩니다.

기타 청구 항목

수강 청구 요약

회차 수에 따라 자동 계산되며, 개별 할인 및 추가 청구 항목을 수정하면 즉시 요약 금액이 바뀝니다.

실제 수강 회차 수 8회
회차 기준 수강료 800,000원
교재비 10,000원
시험비 20,000원
추가 항목 5,000원
금액 할인 -50,000원
비율 할인 -0원
청구 상태 청구 대기
최종 청구 금액 785,000원

수강 등록만 하면 학생 상태는 `청구 대기`로 남고, `수강 등록 및 청구서 발행`을 선택하면 결제 대기로 전환됩니다.

Step 5. 등록 확인

등록 전 마지막으로 학생, 수업, 실제 수강 회차, 청구 금액을 한 번 더 확인합니다.

등록 대상 김학생 / 2025 SAT class

학생 관리와 수업 관리 어느 쪽에서 들어와도 최종 등록 대상은 동일하게 확정됩니다.

회차 및 청구 5회차 ~ 12회차 / 총 8회 수강

자동 기준 금액을 바탕으로 개별 할인과 추가 항목이 모두 반영된 상태입니다.
